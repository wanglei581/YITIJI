## 一、产品三端概览

| 端 | 主要用户 | 主要场景 |
|----|---------|---------|
| 一体机前台（Kiosk） | 求职者 | 简历服务、打印扫描、浏览岗位/招聘会 |
| 管理员后台（Admin） | 运营人员 | 终端管理、订单管理、数据审核、告警处理 |
| 合作机构后台（Partner） | 合作机构工作人员 | 岗位/招聘会数据上传、同步管理、统计查看 |

---

## 二、一体机前台功能清单

### 2.1 首页

| 功能模块 | 说明 | 优先级 |
|---------|------|--------|
| AI简历服务入口 | 跳转 AI 简历子功能 | P0 |
| 打印扫描入口 | 跳转打印/扫描流程 | P0 |
| 岗位信息入口 | 跳转第三方岗位列表 | P0 |
| 招聘会信息入口 | 跳转招聘会列表 | P0 |
| 政策服务入口 | 跳转政策公告 | P1 |
| 设备状态展示 | 打印机在线/离线/故障 | P0 |
| AI助手快捷入口 | 底部固定，全局可达 | P0 |

### 2.2 AI简历服务

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 简历上传 | 上传 PDF/Word | P0 |
| 纸质简历扫描 | 调用打印机扫描，生成 PDF | P1 |
| AI解析 | 提取简历结构化信息 | P0 |
| AI诊断 | 分析简历问题和评分 | P0 |
| 简历优化 | AI 生成优化建议 | P1 |
| 简历打印 | 选择模板 → 预览 → 打印 | P0 |

### 2.3 打印扫描

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 上传文件打印 | 支持 PDF、Word、图片 | P0 |
| 扫码上传 | 手机扫码上传文件 | P1 |
| U盘导入 | 读取 U 盘文件 | P1 |
| 扫描原件 | 调用 ADF 扫描 → PDF | P1 |
| 打印预览 | 展示文件预览和打印参数 | P0 |
| 打印参数设置 | 单双面、份数、彩色/黑白 | P0 |
| 确认打印 | 付款或免费确认 | P0 |
| 打印进度 | 实时展示打印任务状态 | P0 |
| 完成反馈 | 打印完成提示 | P0 |

### 2.4 岗位信息

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 第三方岗位列表 | 来自合作机构的岗位数据 | P0 |
| 岗位搜索/筛选 | 关键词、城市、行业等 | P1 |
| 岗位详情 | 职位描述、要求、来源信息 | P0 |
| 来源机构展示 | 展示 source_name、source_url | P0 |
| 外部跳转 | 去来源平台投递 / 扫码投递 | P0 |
| 收藏岗位 | 用户本地收藏 | P1 |

> **注意**：不得出现"一键投递"、"立即投递"按钮。详见 [compliance-boundary.md](./compliance-boundary.md)。

### 2.5 招聘会信息

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 招聘会列表 | 来自合作机构的招聘会数据 | P0 |
| 招聘会详情 | 时间、地点、参展企业、来源 | P0 |
| 外部预约跳转 | 去来源平台预约 / 扫码预约 | P0 |
| 展位导览 | 如有展位图可展示 | P2 |
| 活动资料打印 | 打印招聘会手册等材料 | P2 |

### 2.6 AI助手

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 自然语言咨询 | 问答模式，支持多轮对话 | P0 |
| 引导功能跳转 | AI 识别意图后推荐跳转路径 | P1 |
| 简历建议 | 结合用户简历给出建议 | P1 |
| 打印帮助 | 引导用户完成打印操作 | P1 |
| 政策问答 | 基于政策文档回答 | P2 |

### 2.7 我的

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 我的简历 | 已上传/解析的简历列表 | P0 |
| 我的文档 | 打印过的文件记录 | P1 |
| 打印订单 | 历史打印记录 | P0 |
| AI服务记录 | 诊断/优化历史 | P1 |
| 外部岗位浏览记录 | 浏览/跳转过的岗位 | P1 |
| 招聘会预约记录 | 跳转过的招聘会 | P1 |
| 账号设置 | 手机号、密码 | P1 |

## 三、管理员后台功能清单

| 模块 | 主要功能 | 优先级 |
|------|---------|--------|
| 工作台 | 关键指标、待处理告警、快捷入口 | P0 |
| 终端管理 | 一体机列表、在线状态、详情 | P0 |
| 打印机管理 | 打印机状态、配置、历史任务 | P0 |
| 外设管理 | 扫码器、摄像头、U盘读取器 | P1 |
| 订单管理 | 打印订单列表、详情、退款 | P0 |
| 文件管理 | 用户文件列表、审查、清理 | P0 |
| AI服务管理 | AI调用记录、用量统计 | P1 |
| 岗位信息源 | 审核/发布来自合作机构的岗位 | P0 |
| 招聘会信息源 | 审核/发布来自合作机构的招聘会 | P0 |
| 合作机构管理 | 机构列表、授权、状态 | P0 |
| 用户管理 | 用户列表、封禁、查看记录 | P1 |
| 告警中心 | 硬件告警、系统告警、处理记录 | P0 |
| 权限管理 | 角色、菜单权限配置 | P1 |
| 日志审计 | 管理员操作日志、文件访问日志 | P0 |

## 四、合作机构后台功能清单

| 模块 | 主要功能 | 优先级 |
|------|---------|--------|
| 工作台 | 数据概览、待审核数、同步状态 | P0 |
| 机构资料 | 机构名称、联系人、授权状态 | P0 |
| 岗位信息管理 | 上传/维护岗位数据 | P0 |
| 招聘会信息管理 | 上传/维护招聘会数据 | P0 |
| 政策公告管理 | 上传政策文件 | P1 |
| 终端数据 | 查看本机构数据在终端的展示情况 | P1 |
| 数据统计 | 浏览量、跳转量、打印量 | P1 |
| 数据源管理 | 配置自动同步数据源 URL | P2 |
| 同步日志 | 数据同步历史记录 | P1 |
| 账号权限 | 子账号管理 | P1 |

> **注意**：合作机构后台不包含候选人管理、简历筛选、面试邀约任何功能。

| P2 | 扩展功能，有时间再做 |

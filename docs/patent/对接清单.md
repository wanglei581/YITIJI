# 对接专利代理机构 — 准备清单

> 给代理机构前先备齐以下内容；带 ❗ 的是他们一定会问、且需要你（公司）先内部确认的。

## 一、交付材料（直接给代理机构）

- [ ] 本目录 5 份技术交底书（`01`–`05` + `README` 专利清单）
- [ ] 各交底书内的 `file_path:line_number` 代码位置（代理人写实施例/附图用）
- [ ] 如有：系统架构图、关键页面截图、设备照片（外观设计用）

## 二、❗ 申请人与发明人（务必先内部确认）

- [ ] **申请人主体**：公司 还是 个人？（决定专利权属）
- [ ] **是否职务发明**：上班期间为完成公司任务所做的发明，申请人通常是公司、发明人是实际开发者。若涉及合伙/外包，提前书面约定权属，避免日后纠纷。
- [ ] **发明人名单**：每件专利对应的实际技术贡献者姓名、身份证号。
- [ ] 申请人为企业的：营业执照、是否**小微企业**（可申请官费减缴）。

## 三、❗ 公开情况自查（关系到新颖性，很重要）

- [ ] 该技术是否已**对外公开**？（上线运营、对外 demo 讲实现原理、发表文章/视频、投标文件公开等）
- [ ] 如已公开，记录**首次公开日期**——可能影响能否申请，须如实告知代理人。
- [ ] 签订含**保密条款**的代理委托合同前，控制技术细节传播。

## 四、选代理机构的要点

- [ ] 优先选有 **计算机/软件/通信** 领域经验的机构与代理人（软件类发明撰写难度高）。
- [ ] 确认是否提供**申请前现有技术检索**（判断新颖性/创造性，避免白报）。
- [ ] 明确报价：代理费 + 官费分别多少、是否含 OA（审查意见）答复、授权后年费谁管理。

## 五、申报优先级建议（来自 README）

| 顺序 | 专利 | 类型 | 理由 |
|------|------|------|------|
| 1 | P1 云打印容错调度 | 发明 | 最技术、撞车概率低，首选 |
| 2 | U1 打印机多维故障诊断 | 实用新型 | 授权快、便宜，先拿一件 |
| 3 | P2 数据源合规接入 | 发明 | 合规差异化卖点 |
| 4 | P4 简历 AI 诊断差量 | 发明 | 算法流程完整 |
| 5 | P3 待机宣传屏调度 | 发明 | 需先检索 |
| 6 | D1 一体机界面 GUI | 外观设计 | 最便宜，占位 |

> 预算有限时：可先报 **P1（发明）+ U1（实用新型）+ D1（外观）** 三件组合，覆盖"方法+装置+外观"三个维度，性价比最高；其余择期。

## 六、地方资助

- [ ] 向公司所在地 **科技局 / 知识产权局** 咨询专利**资助/奖励**政策（申请阶段或授权阶段补贴），可能覆盖大部分费用。

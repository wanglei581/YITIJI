# 交底书 P4：纸质简历的 AI 诊断—优化差量展示与提升度估算方法

> 建议类型：**发明专利** ｜ 创造性：🟢🟢 中
> 技术领域：自然语言处理应用、人机交互、文档智能

---

## 1. 背景技术与现有问题

求职者在自助终端使用纸质简历时，需经历"扫描纸质件 → AI 解析为结构化文本 → AI 诊断 → AI 优化 → 打印"的端到端流程。现有简历工具多为线上单步操作，缺乏：① 面向一体机的多步骤可回退流程编排；② 诊断结果的**可解释可视化**（哪些维度弱、优先改哪些）；③ 优化前后的**字符级差量对比**与**提升度量化估算**；并且常以"提高录用率"等不当承诺误导用户。

## 2. 发明目的

在自助终端上提供一条可解释、可回退、合规的简历诊断—优化流程：以多维评分雷达图呈现诊断结果并自动定位薄弱项，以字符级 diff 展示优化前后差异，以可量化但带免责声明的"提升度估算"辅助决策，避免对录用结果做不当承诺。

## 3. 技术方案（权利要求骨架）

### 独立权利要求（方法）

一种纸质简历的 AI 诊断—优化差量展示方法，包括：

1. **多步骤可回退流程编排**：将简历服务编排为"上传/扫描 → 解析 → 诊断 → 优化 → 打印"的有序步骤，各步骤状态可持久化、可回退重做。

2. **多维诊断与薄弱项自动定位**：对解析后的结构化简历按多个维度评分，以雷达图呈现；自动计算各维度得分率并**取得分率最低的 K 项**作为"优先修改项"并视觉强调，附可执行建议。

3. **优化差量展示**：对优化前后文本执行**字符级 diff**并可视化对比，使用户直观看到被修改/新增/删除的内容。

4. **提升度估算**：基于优化前后各模块的文本变化量（如长度差比），以加权累加映射到一个分段评分区间，输出"预计提升度"数值，并**强制附带免责声明**（"估算值，不代表真实招聘结果"）。

### 从属权利要求方向

- 诊断/优化以可插拔的 AI Provider 抽象实现，Provider 未启用时返回结构一致的占位结果，接入真实 Provider 后无缝切换（便于离线演示与合规灰度）。
- 提升度估算函数对每个模块计算 `ratio = min(1, |afterLen − beforeLen| / beforeLen + 偏置)`，加权累加后映射到限定区间（如 70–95），避免给出绝对化或越界数值。
- diff 可视化支持字符级（word/char diff）开关。

## 4. 关键创新点

| 创新点 | 与惯例对比 |
|--------|-----------|
| 诊断结果**薄弱项自动定位 + 优先级着色** | 多数工具只给总分，不指出改哪里 |
| 优化**字符级 diff** + **量化提升度估算** | 多数工具只给改后文本，无对比无量化 |
| 提升度**强制免责声明**、不承诺录用结果 | 规避对求职结果的不当承诺（合规） |
| Provider 占位—无缝切换 | 便于离线演示与合规灰度 |

## 5. 有益效果

求职者在自助终端获得可解释、可回退的简历诊断—优化体验，明确知道改哪里、改了什么、预计提升多少，同时避免被不当承诺误导。

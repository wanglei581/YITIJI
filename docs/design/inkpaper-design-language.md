# 墨青纸感（Ink-Paper）统一设计语言

> 2026-07-02 定稿。适用范围：Kiosk 一体机前台（fusion-youth 系列）、管理员后台（apps/admin）、合作机构后台（apps/partner）。
> 本文档是三端 UI 铺开与未来真实代码实施的唯一视觉依据；按 CLAUDE.md §9，属设计参考，合规硬约束仍以 docs/compliance/ 为准。
> 原型基准：`.workbuddy/prototypes/fusion-youth-preview-v5.html`（Kiosk）、`admin-console-inkpaper-v1.html`（管理员）、`partner-console-inkpaper-v1.html`（机构）。
> 原型文件属于本地设计参考，不作为仓库交付物；正式落地以本文档和后续明确解冻的 `apps/` 代码改造任务为准。

## 1. 设计立场（为什么是它）

- 一句话：**纸与墨的办事大厅**——政务的可信 + 年轻求职者能接受的温度。
- 三条底线（不可妥协）：
  1. **零 AI 味**：禁紫蓝渐变、深色霓虹光晕、玻璃拟态堆叠、四字对仗营销口号（"高薪之选/精准匹配"类）。
  2. **不伪造**：无真实数据不展示结论；原型假数据必须标注"预览示例"。
  3. **合规做进结构**：边界不是脚注，是产品结构（无发布按钮、外链必填、凭证不回显、无简历字段）。

## 2. 色彩令牌

```css
--paper:   #f4f1e8;  /* 主画布 · 暖米纸 */
--paper-2: #efeadd;  /* 深纸 · 嵌套底/量表槽 */
--surface: #fffdf8;  /* 卡片 · 暖白（禁纯白）*/
--surface-2:#f7f4ec; /* 卡内嵌套 */
--ink:     #10302b;  /* 墨绿 · 标题/侧栏/主按钮 */
--ink-2:   #2c4a43;  /* 次级墨绿 */
--muted:   #5d6b63;  /* 辅助文字（27寸/后台通用下限）*/
--teal:    #1f9e86;  --teal-deep: #157a67;  --teal-soft: #e2f2ec;   /* 品牌青 · 主强调 */
--clay:    #b8683c;  --clay-soft: #f4e8dd;   /* 陶土 · 警告/招聘会 */
--wheat:   #a9781f;  --wheat-soft:#f2ead6;   /* 麦金 · 政策/机构身份 */
--slate:   #3f68b0;  --slate-soft:#e5edf4;   /* 青蓝 · 打印/进行中 */
--plum:    #7a5a86;  --plum-soft: #efe7f1;   /* 梅 · AI/权益（Kiosk 用）*/
--red:     #c14a34;  --red-soft:  #f6e3de;   /* 仅错误/紧急/驳回 */
```

用法规则：teal 是唯一"品牌强调色"；clay/wheat/slate/plum 只做功能分类，禁止整块大面积使用；red 只表达错误与紧急，禁做装饰。

## 3. 三端身份区分（同语言，不同点缀）

| 端 | 侧栏/主色 | 身份点缀 | 备注 |
|---|---|---|---|
| Kiosk 前台 | 深墨绿 Hero 仅首页 | teal | 竖屏 1080×1920，触控 ≥48px |
| 管理员后台 | 墨绿侧栏 | teal（Logo/头像/选中态） | 1440 桌面，26 项 6 组导航 |
| 机构后台 | 墨绿侧栏 | **wheat**（Logo/头像/环境徽） | 12 项 4 组导航，一眼区分身份 |

## 4. 字体

- **标题**（页标题/卡片标题/宋体感场景）：`"Noto Serif SC","Source Han Serif SC","Songti SC","SimSun",serif`，weight 800。
- **正文/按钮/表格**：系统黑体栈（-apple-system / PingFang SC / Microsoft YaHei）。
- 数字一律 `font-variant-numeric: tabular-nums`。
- ⚠️ 真实实施必须把思源宋体字体文件打进应用（Windows 无此字体，回退中易宋体不可接受）。

## 5. 字号与密度

| 场景 | Kiosk | 后台 |
|---|---|---|
| 页标题 | 46/34px 宋体 | 24px 宋体 |
| 卡片标题 | 22–27px 宋体 | 15.5px 宋体 |
| 正文 | 16–17px | 13.5px |
| 表格体/辅助 | ≥15px（触屏下限） | 13px / 11.5px 表头 |

## 6. 组件规范（以原型为准的关键值）

- **卡片**：surface 底 + 1px line-2 边 + shadow-sm；圆角 Kiosk 26/18/14px，后台 14px。
- **主按钮**：ink 底白字；次按钮 surface+1px 边；后台高度 36px（sm 30px），Kiosk ≥56px。
- **状态徽章**：22px 高圆角胶囊 + 前置 6px 圆点；ok=teal / doing=slate / warn=clay / err=red / pend=wheat / off=灰。
- **数据表格**：表头 11.5px 大写间距 + 底边框；行 hover 换 surface-2；行内量表 52×6px。
- **右侧抽屉**：432px，信息栅格(2列) + 宋体分节标题 + 底部操作条；scrim `rgba(16,48,43,.28)+blur(2px)`；260ms cubic-bezier(.2,.8,.24,1)。
- **表单**：label 12.5px + 红星必填；输入框聚焦 teal 边 + 3px `rgba(31,158,134,.14)` 环；涉审计操作必须带"将写入审计日志"hint。
- **筛选**：搜索框 + 状态 Chips（选中 ink 反白 + 数量角标）+ 日期 Chip。
- **Tab**：下划线式，3px teal 圆角条 + 数量角标。
- **分页**：28px 方块，当前页 ink 反白，带省略号。
- **告警卡**：左侧 4px 严重度色条（红/陶土/青蓝），处理后 300ms 滑出消失，角标实时联动。
- **合规边界卡**：wheat-soft 底 + 虚线边 + 盾形图标，用于"不接收简历"等边界声明。
- **Toast**：墨绿 95% 底部胶囊，200ms 上浮入场。
- **触控反馈（Kiosk）**：涟漪（浅色元素 teal、深色元素白）+ `:active scale(0.97)` + 页面切换 280ms 上浮渐入；全部尊重 `prefers-reduced-motion`。

## 7. 图标

- 统一**双色实心**风格（主形 fill + 次要部分 opacity .22–.55），24 viewBox。
- 正式落地接 **iconfont.cn** 自有项目（Symbol 用法）；后台可用 Lucide 但需选实心/双色变体保持一致。
- **禁止 AI 生成图标**；禁止线性/实心混用。

## 8. 形象与照片

- 小青 = **真人照片**（apps/kiosk/public/assets/ai-advisor.png，需肖像授权）+ 通话面板**声纹动画**；禁卡通/机器人形象。
- 照片艺术指导：统一光线与色温、墨绿系背景、圆角矩形裁切。

## 9. 文案

- 功能性描述，不用营销口号；投递/预约文案只用合规白名单（去来源平台投递 / 扫码投递 / 去来源平台预约 / 扫码预约）。
- 未上线能力标"即将上线"且比例受控；未配置模块整卡不渲染（空态只给管理后台）。

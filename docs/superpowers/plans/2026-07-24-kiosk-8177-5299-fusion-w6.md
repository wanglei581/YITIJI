# Kiosk 8177 / 5299 Fusion W6 Integration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 W0–W5 的 86 条生产路由融合换装整合为一个可审查、可构建、可回归的 Kiosk 候选版本，同时诚实保留生产部署和 Windows 真机验收边界。

**Architecture:** W6 不重写业务页面，只负责跨波次集成：注册已存在的 verifier/browser 命令，修复因 CSS 拆分导致的旧守卫漂移，消除嵌套 `main` 语义债，建立 86 路由清单与双视口验收，串行执行 W0–W5 production-build Playwright。所有 API fixture 继续 fail-closed；不删除旧 CSS/import，除非能证明零路由、零 import、零测试和零正式文档依赖。

**Tech Stack:** React 18, React Router 7, TypeScript 5.7 compiler API, Vite 6, Playwright Chromium, pnpm workspace, GitHub Actions.

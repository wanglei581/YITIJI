> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Migrate the 24 profile, activity, toolbox, authentication, legal, screensaver, system-state, notification, and mobile-helper production paths to the frozen fusion presentation while retaining every real member, upload, legal, health-check, timeout, and toolbox behavior.

**Architecture:** Keep `9999d022` W1 primitives immutable and use them only as presentation/accessibility wrappers. Preserve all existing hooks and API modules; replace the two three-line placeholder route components by delegating to existing real capability or by composing existing activity APIs. Split the 907-line detail stylesheet behind its current aggregator so imports stay stable until W6.

**Tech Stack:** React 18, React Router 7, TypeScript 5.7, `@ai-job-print/ui` frozen Kiosk primitives, Vite 6, Node 22 verifier scripts, Playwright 1.54.2 Chromium.
